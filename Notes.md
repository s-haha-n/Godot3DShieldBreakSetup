STEPS TO USE:

1 - open blender clear scene press N 
2 - TICK 'Insert Root' and select FOLDER of skinless animation files
3 - delete all armature except first one confirm action editor has all names of clips
4 - export all default options as gltf
5 - import options in godot Import tab -> Import As : Animation Library set some to loop
6 - load it on humanoid Animation Player node
