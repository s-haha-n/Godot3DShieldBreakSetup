have a set of .fbx files in a folder

Format: FBX Binary, Without Skin, 30fps, no kf red

then use this plugin: https://github.com/Flynsarmy/Mixamo-Root

Open blender, N to panel 'mixamo'

Tick 'insert root'

Animation File: Source - set of .fbx skinless files in folder

delete all but one armature.
check action editor.

export gltf. do not need to reset or reimport in godot automatically adds the new animation to the 
AnimationPlayer object!


---

You can add without the plugin, but will not have ROOT BONE

'animatoins' doesn't have root bone
